在使用本应用时因您的失误出现以下情况均由您承担。

- 个人偏好数据丢失
- 本应用因您的自定义导致的崩溃
- 电池耗尽
- 设备硬件老化或报废
- 您沉迷于在本应用

本应用所有内置链接可能仅供学习与交流使用。如需商用，请自行向相关部门认证并部署 gh-proxy 项目。
